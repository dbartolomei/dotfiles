# System-Wide Claude Code Instructions

## User Context

- Timezone: America/Puerto_Rico (AST)
- Always run `date` (or `TZ=America/Puerto_Rico date`) before generating any date/timestamp output

## Response Style

- Be concise and direct. No filler, no summaries of what you just did.
- Lead with the answer or action, not the reasoning.

## Git Commits

- Use conventional commits: `feat:`, `fix:`, `refactor:`, `docs:`, `chore:`, etc.
- Keep the subject line under 72 characters. Use the body for details.
- **NEVER add `Co-Authored-By` lines.** No credit attribution for AI assistance.
